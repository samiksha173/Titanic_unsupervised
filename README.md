🚢 Titanic Unsupervised Learning

This project applies K-Means Clustering and PCA (Principal Component Analysis) on the Titanic dataset to discover hidden patterns in passenger data.

🚀 Features
🧹 Data cleaning & preprocessing
🔢 Label encoding for categorical data
📊 Feature scaling using StandardScaler
🤖 K-Means clustering (3 clusters)
📉 PCA for dimensionality reduction
📈 Cluster visualization
💾 Export final clustered dataset
🛠️ Technologies Used
Python
Pandas, NumPy
Matplotlib
Scikit-learn
▶️ Run the Project
pip install pandas numpy matplotlib scikit-learn
python your_script_name.py
📊 Output
Cluster distribution
PCA visualization plot
CSV file: titanic_unsupervised_clusters.csv

Understand unsupervised learning
